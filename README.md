# advent24

To install dependencies:

```bash
bun install
```

To run latest:

```bash
bun latest
```

To run specific day:

```bash
bun day :num:
```

To run all days:

```bash
bun all
```

To add new day boilerplate (`export AOC_SESSION_KEY=<session-cookier>` to import your specific input):

```bash
bun next
```

To lint all files with prettier

```bash
bun tidy
```
